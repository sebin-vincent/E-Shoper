# E-Shoper

A microService based application in java

## Services
1. Config Service
2. Discovery Service
3. Gateway Serivce
4. Auth Service
5. Profile Service
6. Invenotry Service
7. Order Service

## Tech Stack
Springboot, SpringCloud, Spring-Data,Spring-Security, Feign, Elastic-stack (ELK for log-analysis),Zipkin, Kafka, Docker, MongoDb
